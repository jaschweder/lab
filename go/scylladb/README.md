# Go + gocql (cassandra/scylladb client)

Example running SQL statements in Golang to ScyllaDB database

### Prerequisites

[Docker](https://docker.com)

```
$ wget -qO- https://get.docker.com/ | sh
```

### Installing

Clone the repo

```
$ git clone https://github.com/jaschweder/lab
```

Go to this folder

```
$ cd ./lab/go/scylladb
```

## Running

```
$ ./run.sh
```
